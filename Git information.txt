Git commands

git --version = Current version of git
git init = creates empty repository for current file.
git status = polls any active file for upload
git add {File or .} = File stages the named file for upload/ . all files will be staged.
git commit -am "insert text" = Will upload any staged files "x" text is for notes. 
git push origin master = loads files for master 
git pull = Pulls current information from master files for all updates.
git push = Updates file.
